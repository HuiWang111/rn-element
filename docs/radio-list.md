## Props
### value
| type | default | required |
| ---- | ---- | ---- |
| string | none | false |

### options
| type | default | required |
| ---- | ---- | ---- |
| string[] or IRadioListOption[] | none | none |

### onChange
| type | default | required |
| ---- | ---- | ---- |
| Function | none | none |

## Interfaces
### IRadioListOption
```ts
interface IRadioListOption {
    label: string;
    value: ReactText;
    disabled?: boolean;
}
```
