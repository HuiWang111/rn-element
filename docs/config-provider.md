# ConfigProvider

## 示例
```tsx
<ConfigProvider showSoftInputOnFocus={false}>
    <List>
        ...
    </List>
</ConfigProvider>
```

## props

### showSoftInputOnFocus
用于设置List.Item组件下的 input 输入框是否在聚焦时需要展示软键盘
| type | default | required |
| ---- | ---- | ---- |
| boolean | true | false |
